# YouTube Volume Control

YouTube Volume Control is a Chrome extension that adds configurable volume preset buttons directly to the YouTube player.

It is designed for touch-friendly and one-tap volume changes and while still working well with keyboard shortcuts.

## Features

- Adds up to 5 volume preset buttons to the YouTube player
- Lets you configure each preset from 0 to 100
- Allows each preset button to be enabled or disabled independently
- Supports keyboard shortcuts for each preset
- Can hide the default YouTube volume slider
- Can hide the default YouTube volume icon
- Highlights the active preset based on the current player volume

## Usage

1. Open any YouTube video page.
2. Use the extension popup to configure preset values and toggle buttons on or off.
3. Click a preset button in the YouTube player to apply that volume.

The extension popup also works as the options page.

## Keyboard Shortcuts

You can assign or change their shortcuts from:

`chrome://extensions/shortcuts`

Each shortcut sets the player volume to the configured preset value for that row.

## Settings

The popup lets you:

- Change the volume value for each preset
- Enable or disable each preset button
- Hide the YouTube volume slider
- Hide the YouTube volume icon

### Reset

Press and hold the reset button for 1 second to clear saved settings and restore defaults.
